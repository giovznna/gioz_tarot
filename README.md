# gioz_tarot
Landing page para agendamentos de Tarot. 🪄

Este projeto é uma landing page desenvolvida para apresentar meus serviços como taróloga e centralizar o agendamento de consultas online.

A página foi pensada para transmitir acolhimento, clareza e intenção: refletindo a forma como conduzo minhas leituras. Nela, apresento quem sou, explico como funcionam as consultas, detalho os serviços oferecidos e compartilho feedbacks reais de consulentes.

✨ Funcionalidades:
- Apresentação dos serviços e valores
- Explicação do processo de atendimento
- Exibição de depoimentos reais
- Formulário de agendamento integrado ao WhatsApp
- Geração automática de mensagem personalizada para contato

🛠️ Tecnologias utilizadas:
- HTML5
- CSS3 (com foco em design sensível e responsivo)
- JavaScript (manipulação do DOM e integração com WhatsApp)

💡 Objetivo:
Criar um espaço simples, bonito e funcional para facilitar o contato com clientes e tornar o processo de agendamento mais fluido e humanizado.
